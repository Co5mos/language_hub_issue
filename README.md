https://marsjpclass.com/

日语学习平台问题收集

一个全面的日语阅读学习平台，采用现代化 Web 技术构建，通过互动文章、测验和词汇管理帮助学习者掌握日语。

## ✨ 核心功能

### 📚 **阅读学习**
- **精选文章**：提供不同难度等级的阅读材料
- **智能文本分析**：点击日文文本即可获得翻译和语法解释
- **逐句分析**：深入解析语法结构和词汇含义
- **音频支持**：标准日语发音，提升听力技能

### 📖 **词汇管理**
- **个人词典**：保存和整理学习中的词汇
- **学习状态**：标记词汇为学习中、复习中或已掌握
- **语境记录**：保存词汇在原文中的使用场景
- **智能复习**：间隔重复记忆法提高学习效果

## 📱 响应式设计

完全响应式设计，适配所有设备：
- **📱 手机端** ：底部导航，单列布局
- **📟 平板端** ：顶部导航，双列布局
- **💻 桌面端** ：完整导航，多列布局
